🚀 QA Bank Monitor

Projeto desenvolvido para simular um ambiente real de testes em sistemas bancários.

💻 Funcionalidades

* Teste automatizado de login
* Simulação de múltiplos usuários
* Detecção automática de erros
* Registro de bugs
* Resposta automatizada ao cliente

🧠 Tecnologias

* Python
* Automação via terminal (CMD)

## 🎯 Objetivo

Simular cenários reais de alta demanda, como em bancos, onde é necessário identificar falhas rapidamente e registrar incidentes.

▶️ Como executar

```bash
py main.py
```
